W5~W6

SSH

1.start.bat輸入sh

2.輸入 ssh-keygen -t rsa -b 4096 -C "學號"

3.找到路徑裡SSH的鑰匙

4.用SCITE把id_rsa.pub 開啟 複製內容

5.到Github-setting-SSH and GPG keys 按"New SSH key"

6.Title打學號 Key貼上剛剛複製的 再按"Add SSH key"

7.打開puttygen.exe

8.按Load

9.到SSH路徑選取id_rsa 選完後按"save private key" 得到ppk 存到同一個路徑

10.打開putty.exe

11.在Saved Sessions 輸入 "github.com" 再按 "Save"

12.到 "Auth" 的"Private key file for authentication" 按"Browse" 

13.選剛剛儲存的ppk

14.再次打開putty.exe 按"github.com" 再按"Save"

15.到自己的倉儲打開 ".git" 用SCITE打開config

16.反白處輸入 "url = git@github.com:學號/對應的資料夾 再把上一行url前加上#號

17.到start.bat裡反白處輸入

set GIT_HOME=%Disk%:\portablegit\bin\
set GIT_SSH=%Disk%:\putty\plink.exe

完成SSH

Develop

https://github.com/mdecourse/cmsimde 的開發, 可以在一個目錄中放入 cmsimde, 然後將 up_dir 中的內容放到與 cmsimde 目錄同位階的地方, 使用 command 進入 cmsimde 目錄, 執行 python wsgi.py, 就可以啟動, 以瀏覽器 https://localhost:9443 就可以連接, 以 admin 作為管理者密碼, 就可以登入維護內容.

cmsimde 的開發採用 Leo Editor, 開啟 cmsimde 目錄中的 cmsimde.leo 就可以進行程式修改, 結束後, 若要保留網際內容, 只要將 cmsimde 外部的內容倒回 up_dir 目錄中即可後續對 cmsimde 遠端倉儲進行改版.

init.py 位於  up_dir 目錄, 可以設定 site_title 與 uwsgi 等變數.
